# Telegram-бот для продажи музыкальных инструментов

Курсовой проект: русскоязычный Telegram-бот с ML-классификацией намерений, поддержкой обычного диалога, использованием `dialogues.txt` и мягким сценарием перевода пользователя к покупке музыкального инструмента.

## Возможности

- обычный вопрос-ответ и small talk;
- классификация намерений через ML (`TfidfVectorizer + LinearSVC`);
- fallback-поиск ответа по `data/dialogues.txt`;
- обработка коротких ответов: `да`, `нет`, `давай`, `ещё`;
- плавный переход от общего общения к теме музыки;
- подбор товара по интересам, уровню и бюджету;
- каталог с несколькими музыкальными инструментами и альтернативами.

## Структура проекта

```text
bot/        # модули бота
data/       # датасеты и каталог товаров
models/     # обученная модель
scripts/    # генерация датасета и обучение
```

## Запуск

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Подготовить датасет

```bash
python scripts/generate_dataset.py
```

Этот шаг создаёт расширенный файл `data/intents_dataset.json` из шаблонов и вариаций, а также пересобирает `data/dialogues.txt`.

### 3. Обучить модель

```bash
python scripts/train_model.py
```

После этого появится файл:

```text
models/intent_classifier.pkl
```

### 4. Добавить токен Telegram

Создать `.env` на основе `.env.example` или выставить переменную окружения:

```env
TELEGRAM_BOT_TOKEN=ваш_токен
```

### 5. Запустить бота

```bash
python -m bot.main
```

## Логика продаж

Бот не сразу рекламирует товар. Сначала он:

1. поддерживает обычную беседу;
2. переводит разговор к интересам и музыке;
3. выясняет опыт пользователя;
4. спрашивает предпочтения и бюджет;
5. рекомендует подходящий инструмент;
6. при отказе предлагает альтернативу.

## Примечания по данным

- `scripts/generate_dataset.py` генерирует расширенные примеры интентов автоматически;
- в датасет включены короткие ответы вроде `да`, `нет`, `давай`, `ещё`;
- `dialogues.txt` тоже генерируется и дополняется большим количеством шаблонных диалоговых пар, чтобы бот мог лучше поддерживать обычное общение.

## Основные файлы

- `bot/nlp.py` — очистка и нормализация текста;
- `bot/intent_model.py` — обучение и предсказание интентов;
- `bot/dialogue_retrieval.py` — поиск ответа по `dialogues.txt`;
- `bot/sales_funnel.py` — сценарий мягкой продажи;
- `bot/bot_engine.py` — объединение всех механизмов;
- `data/products.json` — каталог товаров.
